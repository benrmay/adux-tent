# Architeuthis Dux Tent Case

## What is this?

A case/tent for for [Architeuthis Dux by Tapi][1].

## Usage

* Install OpenSCAD
* Open adux-tent.scad
* Edit `tent_angle` to suit your needs.
* Render (F6)
* Export STL (F7)
* Print the stl

[2]: https://github.com/tapioki/cephalopoda/tree/main/Architeuthis%20dux
